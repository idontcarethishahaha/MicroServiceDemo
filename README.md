# MicroServiceDemo
A learning project for Spring Cloud Alibaba microservices technology, recording microservices development practices and core principles.

[![Spring Cloud](https://img.shields.io/badge/Spring%20Cloud-Alibaba-green.svg)](https://spring.io/projects/spring-cloud-alibaba)
[![Java](https://img.shields.io/badge/Java-17-orange.svg)](https://www.oracle.com/java/)
[![Microservices](https://img.shields.io/badge/Architecture-Microservices-blue.svg)](https://microservices.io/)

## 🚀 Spring Cloud Alibaba 微服务技术学习仓库
本项目聚焦微服务架构核心技术，基于 Spring Cloud Alibaba 生态，系统学习并实践微服务全链路开发：

1.核心组件：Nacos 服务注册发现与配置中心、Sentinel 熔断限流、Seata 分布式事务、Gateway 网关  
2.技术实践：微服务拆分、服务调用、负载均衡、链路追踪、容器化部署  
3.学习记录：原理笔记、实战代码、项目踩坑与优化方案  
4.用于构建完整微服务技术栈，为后端开发求职与项目实践提供参考。
## 🚀 Spring Cloud Alibaba Microservices Learning Repository
This project focuses on the core technologies of microservice architecture, based on the Spring Cloud Alibaba ecosystem, systematically learning and practicing full-link microservice development:

1.Core components: Nacos service registration & discovery/configuration center, Sentinel circuit breaker & rate limiting, Seata distributed transaction, Gateway  
2.Technical practices: microservice splitting, service invocation, load balancing, distributed tracing, containerized deployment  
3.Learning records: principle notes, practical code, project pitfalls and optimization solutions  
4.Used to build a complete microservices technology stack, providing reference for backend development job hunting and project practice.
